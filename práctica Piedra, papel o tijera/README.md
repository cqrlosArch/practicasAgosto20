# GAME: Rock, papper, scissor, spock, lizard

## Workflow Webpack: Babel, Sass, Pug, Images, Files & Browser-Sync

## Start Proyect: 
- Para desarrollo: 
```bash
    ``npm run start`` ó ``yarn start``
```
- Para producción: 
```bash
    ``npm run build`` ó ``yarn build``
```
